# Ruby APP

Simple page view log parser.

## Requirements

* [Ruby 2.6.5](https://www.ruby-lang.org/en/documentation/installation/)

## Development
```
$ gem intall bundler
$ bundle install
```
### Run tests
```
$ bundle exec rspec
```

### Linting
```
$ bundle exec rubocop
```

### EXEC Command

ruby main.rb filepath
